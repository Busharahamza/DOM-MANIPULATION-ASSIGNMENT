# DOM-MANIPULATION-ASSIGNMENT
Student Registration System
A responsive Student Registration System created using HTML, CSS, and JavaScript, enabling users to manage student records with add, view, edit, and delete functionality, leveraging Local Storage for persistent data retention across page reloads.

Its feature includes 
Adding new student records like Name, Student ID, Email and Contact Number.
Edit & delete existing records.
Responsive layout for mobile, tablet, and desktop.
A table for registered students.

Languages used:
HTML5
CSS3
JavaScript
